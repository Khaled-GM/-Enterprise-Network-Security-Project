# -Enterprise-Network-Security-Project
- Designed and implemented a secure enterprise network using subnetting and IP addressing
- Configured OSPF routing with authentication and implemented VLAN segmentation
- Secured network access using SSH, AAA (RADIUS, TACACS+)
- Applied advanced security measures including ACLs, Zone-Based Firewall, and VPN
- Configured monitoring services (Syslog, NTP) to enhance network visibility
